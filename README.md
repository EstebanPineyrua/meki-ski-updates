## ¿Qué hace?

### Órdenes de trabajo
Creá órdenes en segundos, asigná reparaciones por equipo y seguí el estado de cada trabajo: **En proceso → Listo → Entregado**. Al crear una orden se imprime el ticket automáticamente.

### Clientes
Base de datos de clientes con historial completo de órdenes anteriores y total gastado. El sistema recuerda el teléfono y email para no tener que tipearlos de nuevo.

### Seña y pagos
Registrá señas parciales al recibir el equipo y llevá el control del saldo pendiente por orden.

### Inventario
Controlá el stock de productos con alertas de stock mínimo.

### Notificaciones
Alertas automáticas de órdenes vencidas, entregas del día y stock bajo — todo en un solo panel.

### Reportes
Estadísticas de órdenes, reparaciones más frecuentes e ingresos. Exportables a PDF y Excel.

### Emails automáticos
Cuando una orden ingresa, está lista o fue entregada, el cliente recibe un email automático con el detalle.

### Actualizaciones automáticas
La app se actualiza sola. No necesitás desinstalar ni volver a descargar manualmente.
